Reinforcment Learning Project

We implemented a Q-Learning algorithm to find the shortest points between two locations on campus. Q-Learning is a Model-free RL algorithm, meaning it does not need a representation of the enviroment to find the optimal policy. We interacted with the Open Street Maps api through the overpassQuery language to extract transitions and distances.
